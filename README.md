# After Class

After Class is an educational website focused on sexual health, pleasure, and confidence for adults.

## Concept
Things your school never taught you about sex — explained clearly, calmly, and without shame.

## Features
- Editorial-style educational articles
- Golden ratio–based layout and typography
- Inclusive, non-explicit, 18+ educational content
- Clean, scalable HTML & CSS structure

## Tech Stack
- HTML5
- CSS3
- Google Fonts

## Status
Version 1 — Content-first educational platform
